Take a model and place all of its parts on layers.

![[Ray Gun]](/examples/raygun-layers.png)

## Try this prompt

{{< prompt >}}
Assign every object in this file to an appropriate layer.
{{< /prompt >}}

## What you should see

The assistant will evaluate the model, choose groupings and then place each group on individual layers.
