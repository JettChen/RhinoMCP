Your assistant can create new Grasshopper definitions for you, using a mix of components and scripts.

## Try this prompt

{{< prompt >}}
Create a Grasshopper script that generates a twisty tower.
{{< /prompt >}}

## What you should see

The assistant opens Grasshopper (if it isn't already), drops in the right components, connects them, and solves the canvas. Likely the first script will fail, and the Agent will check the canvas and iterate until correct.

Once it is done, you can grab the slider, change values, and watch it update live in Rhino.
